
# Jogo da forca 
## Como compilar

1. Em projeto, vá até a pasta src (cd src)
2. Digite na linha de comando "g++ main.cpp -o main.exe"
3. Para rodar, basta digitar o seguinte comando "./main.exe"
4. Pronto, agora só basta jogar!
